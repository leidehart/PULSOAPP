# PULSOAPP — pacote de melhoria pronto para aplicar

Este pacote substitui o arquivo `index.html` do repositório por uma versão melhorada do app.

## O que foi melhorado

- Exercícios com perguntas variadas.
- Alternativas embaralhadas a cada sessão.
- A resposta correta não fica sempre na mesma posição.
- Avaliação final ao terminar o exercício.
- Leitura do momento emocional com base no check-in.
- Contagem real de dias ativos.
- Contagem de dias em que exercícios foram concluídos.
- Sequência geral de uso.
- Sequência de exercícios.
- Calendário de 30 dias mostrando interações.
- Escrita terapêutica guiada com temas variados.
- Resumo copiável para levar como reflexão para a terapia.

## Arquivo principal

Use o arquivo:

`index.html`

Ele deve substituir o `index.html` atual do repositório.

## Como aplicar pelo site do GitHub

1. Entre no repositório: `leidehart/PULSOAPP`.
2. Clique no arquivo `index.html`.
3. Clique no ícone de lápis para editar.
4. Apague todo o conteúdo atual.
5. Abra o `index.html` deste pacote no seu computador.
6. Copie tudo e cole no editor do GitHub.
7. No final da página, escreva a mensagem do commit:

   `Melhora exercícios, avaliação final e evolução real do PulsoApp`

8. Clique em **Commit changes**.

## Como aplicar por upload no GitHub

1. Entre no repositório `PULSOAPP`.
2. Clique em **Add file** > **Upload files**.
3. Arraste o `index.html` deste pacote.
4. Confirme que ele vai substituir o arquivo atual.
5. Faça o commit.

## Como aplicar pelo computador com Git

```bash
git clone https://github.com/leidehart/PULSOAPP.git
cd PULSOAPP
cp /caminho/do/pacote/index.html ./index.html
git add index.html
git commit -m "Melhora exercícios, avaliação final e evolução real do PulsoApp"
git push
```

O app faz leituras de autoconsciência e apoio emocional, mas não substitui psicoterapia, avaliação clínica ou atendimento profissional.
